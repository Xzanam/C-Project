Input Guide:-
	The computer will prompt you for an input as shown below after you select to start a game in the main menu.
	Your Input:a- 
The syntax for user input is;
	Your Input:a-'position' 'value'
For e.g
	Your Input:a-11 2 
	inserts the value 2 at position 11 i.e first row, first column. 


Game Rules:-

Sudoku is played on a grid of 9 x 9 spaces. Within the rows and columns are 9 "squares" 
(made up of 3 x 3 spaces). Each row, column and square (9 spaces each) needs to be filled out with the numbers 1-9, without repeating any numbers within the row, column or square.
